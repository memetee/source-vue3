在这个里面，我们通过 reactive 创建出 state，然后把 state 结构之后，拿到 name 了，但是修改了 state.name，effect 没有执行
如果想让这个 name 是响应式的，怎么办呢？
在官方中是通过 toRef 可以实现的，接着来实现一下。

packages\reactivity\src\ref.ts

```ts
	...

/**
 * toRef 的内部实现类，只是一个对原对象的代理
 */
class ObjectRefImpl {
  [ReactiveFlags.IS_REF] = true // 他是一个ref
  constructor(
    public _object,
    public _key,
  ) {}
  get value() {
    return this._object[this._key]
  }
  set value(newValue) {
    this._object[this._key] = newValue
  }
}

/**
 * 将 reactive 对象的某个属性解构成 ref
 * 场景：const name = toRef(state, 'name')，这样修改 name.value 也会影响 state.name
 */
export function toRef(target, key) {
  return new ObjectRefImpl(target, key)
}

```

## toRefs

toRefs 的使用

packages\reactivity\src\ref.ts

```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
  </head>
  <body>
    <div id="app"></div>
    <script type="module">
      import {
        reactive,
        ref,
        effect,
        computed,
        watch,
        toRef,
        toRefs,
      } from '../dist/reactivity.esm.js'

      const state = reactive({
        name: '张三',
        age: 18,
      })

      let { name } = toRefs(state)
      effect(() => {
        console.log('name =>', name.value)
      })

      setTimeout(() => {
        state.name = '李四'
      }, 1000)
    </script>
  </body>
</html>
```

toRefs 的作用是拿到响应式对象后，能结构所有的属性，并且这些属性都支持响应式。

```ts
/**
 * 将整个 reactive 对象的所有属性转为 ref
 * 场景：const { name, age } = toRefs(state)，常用于 setup 函数的解构返回
 */
export function toRefs(target) {
  const res = {}

  for (const key in target) {
    res[key] = new ObjectRefImpl(target, key)
  }
  return res // name => ObjectRefImpl, age => ObjectRefImpl
}
```

这里想要访问 name 的值，必须使用 .value 来访问，那我如果不想加 .value 而是直接访问 name 呢？

packages\reactivity\src\ref.ts

```ts
	...

export function unref(value) {
  return isRef(value) ? value.value : value
}

export function proxyRefs(target) {
  return new Proxy(target, {
    get(target, key, receiver) {
      /**
       * 自动解包ref
       * 如果这个 target[key] 是一个ref，那就返回 ref.value, 否则返回 target[key]
       */
      const res = Reflect.get(target, key, receiver)

      return unref(res)
    },
    set(target, key, newValue, receiver) {
      const oldValue = target[key]

      /**
       * 如果 target.a 是一个ref，那么就直接把值给他，不要让他 .value
       * 如果更新了 state.a 它之前是一个 ref, 那么会修改原始的 ref.value 的值 等于 newValue
       * 如果 newValue 是一个ref，那就算了 （ state.a=ref(1) ）
       */
      if (isRef(oldValue) && !isRef(newValue)) {
        /**
         * const a = ref(0)
         * target = {a: a}
         * 更新target.a = 1, 它就等于更新了 a.value
         * a.value = 1
         */
        oldValue.value = newValue
        return true // 这里不用走下面的trigger，因为oldValue是一个ref，它发生了修改会自动触发ref中的setue
      }

      return Reflect.set(target, key, newValue, receiver)
    },
  })
}
```

重新使用

```ts
const state = reactive({
  name: '张三',
  age: 18,
})

const setupState = proxyRefs({
  ...toRefs(state),
})

effect(() => {
  console.log('setupState.name =>', setupState.name)
})

setTimeout(() => {
  setupState.name = '李四'
}, 1000)
```
